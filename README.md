# SPA Test Task (Laravel + Vue 3)

Тестовое задание: Реализация SPA с формой и сохранением данных с использованием паттернов проектирования.

## Архитектура решения

**Backend (Laravel):**
* Применен паттерн **Factory Method** (Фабрика). 
* Создан `SaverInterface` и два класса-реализации (`DatabaseSaver`, `EmailSaver`). 
* Контроллер очищен от бизнес-логики: он делегирует сохранение данных нужной реализации через `SaverFactory` в зависимости от выбора пользователя. Удовлетворяет принципам SOLID (OCP, DIP).

**Frontend (Vue 3 + Composition API):**
* Настроено SPA-приложение через встроенный сборщик **Vite**.
* **Vue-Router:** Реализована навигация без перезагрузки страницы (Форма -> Результат).
* **Vuex:** Реализован паттерн State Management. Данные из формы отправляются через `Action` (Axios POST), а после успешного ответа сервера сохраняются в глобальный State через `Mutation` и выводятся на странице результатов через `Getters`.

## Как запустить

1. Клонировать репозиторий.
2. Установить PHP-зависимости: `composer install`
3. Установить JS-зависимости: `npm install`
4. Скопировать `.env.example` в `.env` и сгенерировать ключ: `php artisan key:generate`
5. В первом терминале запустить бэкенд: `php artisan serve`
6. Во втором терминале запустить сборщик фронтенда: `npm run dev`
7. Открыть `http://127.0.0.1:8000` в браузере.
